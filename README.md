A responsive YouTube-style video browsing interface for the Web Dev Cohort 2026 assignment.

## Features

- Fetches live video data from the FreeAPI YouTube videos endpoint
- Responsive card grid with thumbnails, titles, duration, channel name, views, date, description, and tags
- Search and topic filters
- Load more pagination
- Lightweight plain HTML, CSS, and JavaScript

## API

```txt
https://api.freeapi.app/api/v1/public/youtube/videos
```

## Run locally

Open `index.html` in a browser, or serve the folder with any static server:

```bash
npx serve .
```
